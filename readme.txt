Simple plugin to create a gallery. It also allows upload from the front-end.

== Description ==

Event Stream Gallery is a gallery plugin which allows live uploading of images via multiple people at the same time. Send your custom upload URL to all of your friends, family and co-workers and upload images to the same gallery at the same time from Desktop, PC, Tablet and Mobile Phones!

Create both gallery pages and upload pages with simple shortcodes with handy variables. Event Stream Gallery is perfect for weddings, sports games, family gatherings and many other events. 

= Features =

* Easy to use
* Front-end upload images
* Password protected upload
* Unlimited Galleries
* Create a new gallery in seconds
* Gallery page shortcode
* Gallery upload Shortcode
* Multiple users can contribute
* Instantly populate your gallery from anywhere!
* WordPress login not necessary
* Scalable: You can have thousands of images per gallery which are saved in gallery subfolders which improves the loading.

The Pro Addon will be available soon.

== Installation ==

1. Search within WordPress for Event Stream Gallery or download and unzip the package, and upload `event-stream-gallery` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
